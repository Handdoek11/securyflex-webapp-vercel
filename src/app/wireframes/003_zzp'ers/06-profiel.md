# ZZP Profiel - Profile, Certificates & Settings

## Overzicht
Het Profiel scherm combineert persoonlijke informatie, professionele certificaten en instellingen op één centrale plek. ZZP'ers kunnen hier hun profiel beheren, documenten uploaden, en app-voorkeuren aanpassen.

## Functionaliteit
- **Profiel beheer**: Persoonlijke informatie en professionele details
- **Certificaten**: Upload en beheer van vereiste documenten
- **Notificatie instellingen**: Granular control over alerts en berichten
- **Privacy & Veiligheid**: GDPR compliance en data controle
- **App voorkeuren**: Taal, thema, en gebruikerservaring
- **Account beheer**: Beveiliging, uitloggen, account verwijderen

## Key Features
- **Document management**: Secure upload van certificaten met vervaldatum tracking
- **Privacy controls**: Gebruiker controle over data visibility
- **Notification preferences**: Detailed control over push notifications
- **Security settings**: 2FA, password changes, login history
- **GDPR compliance**: Data export, deletion requests, consent management

### Settings API Endpoints
```typescript
// Get user settings
GET /api/zzp/settings
{
  settings: UserSettings,
  lastUpdated: Date
}

// Update settings
PUT /api/zzp/settings
{
  settings: Partial<UserSettings>
}

// Upload certificate
POST /api/zzp/certificates
FormData: file, type, expiryDate

// Export user data
POST /api/zzp/data-export
{
  dataTypes: string[],
  format: 'json' | 'csv' | 'pdf',
  dateRange: { start: Date, end: Date }
}

// Delete account
DELETE /api/zzp/account
{
  confirmation: string,
  reason?: string
}
```

### Database Schema Used
- **users** (profile information, preferences)
- **securityProfessionals** (professional details, certificates)
- **userSettings** (app preferences, notification settings)
- **fileUploads** (certificates and profile photos)
- **auditLogs** (settings changes, security events)
