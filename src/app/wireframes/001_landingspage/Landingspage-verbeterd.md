## TECHNISCHE SPECIFICATIES

### Performance Requirements
- **FCP**: < 1.5s
- **LCP**: < 2.5s  
- **CLS**: < 0.1
- **TTI**: < 3.5s

### Critical Rendering Path
1. Inline critical CSS
2. Preload hero image/video
3. Lazy load below-fold content
4. WebP images met fallback
5. Service worker voor offline

### Analytics Events
- Hero CTA click
- Video play
- Calculator interaction
- Shift application
- Registration start/complete
- Chat initiation

### A/B Test Varianten
1. Video vs Static image hero
2. €4,99 vs 14 dagen gratis focus
3. Live shifts boven vs onder fold
4. Calculator prominent vs subtle
5. Single vs Triple CTA hero

## CONVERSIE PSYCHOLOGIE

### Persuasion Triggers
1. **Urgency**: Live shifts, limited spots
2. **Social Proof**: 2847 users, testimonials
3. **Authority**: WPBR, ISO, partners
4. **Reciprocity**: Free trial period
5. **Scarcity**: Black Friday deal

### Friction Reducers
1. No credit card for trial
2. 5-minute onboarding
3. VOG later allowed (30 days)
4. Multiple payment methods
5. Cancel anytime
6. 24/7 WhatsApp support

### Trust Builders
1. Security badges prominent
2. Real user photos in reviews
3. Company logos (AH, Schiphol)
4. Bank-level encryption
5. GDPR compliance visible
6. KvK number in footer
